> 💡아나콘다 기본 세팅(vs code 연결)
> 
1. 아나콘다 프롬프트(anaconda prompt) 열기
2. 프롬프트 창 맨 앞에 (base)라고 작성되어 있는지 확인(이게 기본)
3. 가상환경 생성: AI 개발용 가상 환경 생성
    - conda create -n ai_study python=3.10 -y
    - ai_study라는 이름의 방(-n ai_study)을 만들고, 거기에 파이썬 3.10 버전(python=3.10)을 설치
    - -y를 선택하면 알아서 설치
4. vs code 켜기
5. 화면에서 VS Code 화면에서 키보드의 Ctrl + Shift + P 를 동시에 입력
6. 그 검색창에 Python: Select Interpreter 라고 치고 엔터
7. 우리가 아까 만든 ai_study 라는 이름과 conda 라는 글자가 같이 적혀 있는 줄을 찾아서 마우스로 클릭 (예시 : Python 3.10.x ('ai_study': conda))
8. 성공 확인:
    - 키보드의 Ctrl + Shift + ` 입력
    - 화면 아래쪽에 까만 창(터미널)이 열림
    - 까만 창에 글씨가 뜰 텐데, 맨 앞에 (ai_study) 라고 적혀 있다면 연결 성공
9. 이제 VS Code에서 파이썬 파일(.py)을 만들고 코드를 짜면, 컴퓨터는 자동으로 아나콘다의 ai_study 방에 있는 파이썬을 불러와서 일을 처리함. 언제든 새로운 방을 만들면 2단계 방법(Ctrl+Shift+P)을 써서 방을 이리저리 옮겨 다니며 코딩 가능

> 💡변수
> 
1. 어떤 값이든 담을 수 있는 확정되지 않은 값이라 생각하면 편함

    - int(3, 4, 5)
    - float(3.4, 5.7, 실수형)

    - ""로 감싼 문자
    - ""로 감싸면 숫자라도 무조건 다 문자열 변수
    - 문자열 더하기, 곱하기 등 사칙연산도 가능(예 : "hi" + "name") / 단 문자열 곱하기는 문자열 * 숫자만 가능
    - 인덱싱 기능 : a = "hello"란 문자열 변수가 있다면 a[0], a[1]와 같이 문자열 인덱싱으로 해당 문자만 가져 올수 있으며, a[0:2]처럼 슬라이싱도 가능하다
    - 문자열 포맷팅은 C언어와 동일
    - name = "Lee" / age = 30 / one = f"My name is {name} and I am {age} years old." --> 가장 최신의 문법
4. 리스트 자료형

    - 인덱싱 기능은 문자열과 같다
    - 리스트도 더하기, 곱하기 등 연산이 가능하다
    - 리스트에서 사용 가능한 다양한 함수가 존재하니 참고
5. 딕셔너리 자료형
    - 사전과 같이 대응되는 두개의 값이 있는 자료형을 말한다
    - {key1: value1, key2: value2, key3: value3} --> 기본형태
    - 자료를 더하고 사라지고 대신하는 방식은 코드 예제 참고
    - 주의 사항:
        - 같은 키값을 사용하지 말아야 한다

> 💡조건문
> 
1. 어떤 값이 참일때와 거짓일때 나눠서 따로 로직이 이루어지는 문법
2. 주로 if문
    - if 변수가 참일때, 변수가 거짓일때
    - 코드 예제 참고
3. 
```
    if 조건문:
        수행할 문장1
        수행할 문장2
    else문:
        수행할 문장1
        수행할 문장2
```
4. if 조건문은 들여쓰기가 굉장히 중요하다. 코드 흐름이 들여써서 내부에 있다는게 명확해야 한다
5. 조건문에 쓰이는 연산자
    - 비교연산자(<, >, ==, !=, >=, <=)
    - and, or, not (and는 두 조건이 모두 참 / or는 둘중 하나만 참이면 true 로직 / not은 조건이 참이 아닐때 true 로직)
    - in, not in (리스트, 문자열 등에서 그 안에 특정값이 포함되어 있는지 확인)
    - pass (아무것도 하고 싶지 않을때 사용)
6. elif : 여러가지 조건을 판단하는 구문

> 💡반복문
> 
1. 같은 작업을 여러번 반복할때 쓰는 구문
2. 반복문은 while과 for 반복문이 있다
    - while 반복문 : 어떤 조건이 참일때 계속 같은 로직이 반복적으로 되는 반복문
    - for : 지정한 여러개의 리스트에서 뽑아서 사용할때 사용하는 반복문
3. while 조건문:
```
    while 조건문:
        수행할 문장
        ----
        ----
        break #강제로 반복문을 빠져나오는 코드
        continue #계속 반복문을 반복한다는 코드
```
4. 무한반복루프 : while True: 구문 / 종료 조건을 준다
5. for 조건문:
```
    for 변수 in 리스트(튜플, 문자열):
        수행할 문장
        ----
        ----
```

>

    - 프로그래밍을 하다보면 반복 작업이 많아지는데, 그런 로직을 하나로 묶어 사용하기 위해
    - 기능 단위로 코드를 짜기 위해
4. 함수 선언 방법:
```
    def 함수_이름(매개변수):
        수행할 문장
        ----
        ----

```
5. 용어 정리:
```
    def add(a, b):
        return a+b
    
    add(3,4)

    # 3, 4는 인수로 함수를 호출할때 넣은 변수
```

    - 예시 : add = lambda a, b: a + b

> 💡Numpy
> 
1. Numpy : 다차원 배열을 빠르게 분석하는 파이썬 라이브러리. 대부분의 복잡한 수학 배열 객체를 분석하는데 필수적인 라이브러리
2. numpy는 보통 파이썬 기본 내장 라이브러리에 있지만 만약 없다면 패키치 설치를 하면 됀다(설치하는 방식은 노션 정리 참고)
3. list와 np.array의 차이:
    - 리스트는 다른 타입의 변수가 다 섞일 수 있지만 np는 그게 불가능하다
    - 수학 연산에서 다른 타입의 변수가 들어가는건 굉장히 비효율적이고 메모리를 많이 사용한다
    - 데이터 유형 메커니즘이 최적화 되어 있다
