1. 데이터 분석, 머신러닝

- 가상환경 제작시 자체 제작 방식으로 진행한다

> 💡아나콘다 설치하기
> 
1. 아나콘다 다운로드(https://www.anaconda.com/download)
2. 설치 시 주의할점!!!!
    1. (Add Anaconda3 to my PATH…) 체크박스가 보일텐데 무조건 체크 해제된 상태로 두자
    2. (Register Anaconda3 as my default…) 체크박스가 다음으로 보일텐데 이건 체크된 상태로 두자
3. 설치 후 anaconda navigator로 들어가서 확인
4. 아나콘다를 쓰는 이유 : 개발 시 가장 중요한것은 가상 환경 구성이다. 가상 환경 구성을 하는 이유는 복잡한 개발을 하는데 한 공간에서 모든 작업을 하면 서로 꼬이는 문제가 생긴다. 그러므로 전용 공간을 구성해줘야 하는데 그런 작업에 가상환경이 필요하다. 아나콘다는 이런 가상환경 구성에 매우 편리하다
